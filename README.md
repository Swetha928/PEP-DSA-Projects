# Diabetes Classification using Machine Learning

## Project Overview
This project predicts whether a person has diabetes using medical attributes such as glucose level, BMI, insulin, and age.

## Dataset
Pima Indians Diabetes Dataset

## Algorithm Used
Logistic Regression

## Steps Followed
- Data loading and preprocessing
- Train-test split
- Model training using Logistic Regression
- Model evaluation using accuracy score

## Result
The model achieved an accuracy of approximately 74%.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab
